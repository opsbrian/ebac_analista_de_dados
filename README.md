# ebac_analista_de_dados

Para executar o arquivo, verifique se o mesmo pode ser executado. 

Insira o comando: 
l
Se o arquivo aparecer em verde, significa que ele pode ser executado, se não estiver, cole o comando abaixo no terminal:
chmod +x calculadora.sh
chmod +x calculadora.py

Agora o arquivo deve estar em ver e pronto para ser executado.
Use o comando para executar:
./calculadora.sh
